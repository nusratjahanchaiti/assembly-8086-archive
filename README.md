# assembly-8086-archive
*assembly-8086-archive* is a collection of assembly language programs written for practice and learning low-level programming concepts.  
This repository focuses on 8086 assembly and covers arithmetic operations, stack usage, string manipulation, loops, and bitwise logic.

## Contents
- Arithmetic operations (Add, Subtract, Multiply, Divide)
- Stack-based infix to postfix expression evaluation
- String reverse programs
- Array processing (largest/smallest number)
- Bitwise operations (even/odd checking)
- Prime number check
- Lower to Upper case
- Mathematical programs (factorial,fibonacci)
- Sorting
- 5x5 Matrix
- Largest Number
- Addressing Modes
- Alphabet Printing A to Z

## Tools
- 8086 Emulator
- MASM / TASM compatible syntax

## Purpose
This repository is created for learning, experimentation, and strengthening understanding of assembly language concepts.
